Project Doctor

Diagnose. Clean. Optimize.

02

Installation

1. Install Project Doctor from Fab.
2. Enable the plugin in Unreal Engine.
3. Restart the editor.
4. Click the Project Doctor toolbar button.

Quick Start

1. Open Project Doctor.
2. Click Run Scan.
3. Review Errors, Warnings and Info items.
4. Double-click results to open assets.
5. Use cleanup tools to resolve issues.
6. Run another scan to verify improvements.

Cleanup Tools

Fix Redirectors

Repairs redirector assets generated by asset moves and renames.

Delete Empty Folders

Removes unused content folders and helps keep projects organized.

Fix Naming Issues

Automatically renames assets to follow common Unreal Engine naming conventions.

Results Navigation

  • Double-click a result to open the associated asset.
  • Browse directly to assets in the Content Browser.
  • Copy asset paths and issue details.
  • Filter results by severity.
ERROR Broken references, missing assets or project issues that should be fixed first.
WARNING Potential maintenance problems such as naming inconsistencies or unused folders.
INFO Helpful notes for project structure, organization and cleanup opportunities.

Supported Version

Project Doctor is currently tested and supported on Unreal Engine 5.7.

Important Notes

Always use source control before performing automated cleanup operations. Project Doctor modifies project content when cleanup tools are executed.
