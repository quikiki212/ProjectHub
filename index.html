Project Hub

Find. Favorite. Launch.

Project Hub is a lightweight productivity dashboard for Unreal Engine. It helps developers quickly find, favorite, reopen, and manage the assets they use most without repeatedly digging through the Content Browser.

Quick Start Asset Filters
Smart Search Find project assets faster
Favorites Pin important assets
Usage History Recent and most-used assets

Asset Search

  • Project-wide asset search
  • Blueprint-prioritized ranking
  • Exact-match boosting
  • Asset type filtering
  • Generated asset filtering

Workflow Tools

  • Favorites
  • Recently Opened
  • Most Used Assets
  • Persistent usage data
  • Double-click asset opening

Editor Integration

  • Dockable Window
  • Native Asset Icons
  • Content Browser Friendly
  • Editor-only Plugin
  • No Runtime Impact

Why Use Project Hub?

01
Search instantly

Use the search bar to quickly find Blueprints, Materials, Meshes, Maps, UI assets and more.

02
Open assets faster

Double-click any result to open it directly in the appropriate Unreal asset editor.

03
Favorite key assets

Pin the assets you use constantly so they are always available in one place.

04
Return to your workflow

Use Recently Opened and Most Used to quickly resume work without searching again.

Installation

1. Install Project Hub from Fab.
2. Enable the plugin in Unreal Engine.
3. Restart the editor.
4. Open Project Hub from the Tools menu.

Quick Start

1. Open Project Hub.
2. Type into the search bar.
3. Use the asset type dropdown to narrow results.
4. Double-click a result to open it.
5. Click Favorite to pin important assets.
6. Use Recently Opened and Most Used to return to active work.

Asset Filters

Common Assets

Filter by Blueprints, Materials, Meshes, Textures, Animations, Audio, Niagara, UI, Maps, Data Assets, Input Assets and Sequences.

Cleaner Results

Project Hub filters out engine content, redirectors, external actors and generated assets to keep search results focused.

Smart Ranking

Blueprints, exact matches, favorites, recently opened and frequently used assets are prioritized automatically.

Asset Navigation

  • Double-click assets to open them directly.
  • Favorite important assets from search results.
  • Right-click favorites to remove them.
  • Track recently opened assets automatically.
  • Surface most-used assets based on real editor usage.
SEARCH Find project assets from a single dockable window.
FILTER Narrow results by common Unreal Engine asset types.
FOCUS Spend less time navigating folders and more time building.

Supported Version

Project Hub is currently tested and supported on Unreal Engine 5.7.

Important Notes

Project Hub is an editor-only productivity plugin. It does not affect packaged builds and stores usage data locally inside the project.
