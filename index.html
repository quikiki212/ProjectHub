Project Hub

Find. Favorite. Launch.

A lightweight Unreal Engine editor productivity tool for quickly finding, opening, and returning to the assets you actually use.

Features Quick Start Asset Filters Notes

Core Features

Fast Asset Search

Search project assets instantly with smart ranking and Blueprint-prioritized results.

Favorites

Pin important assets and access them from a persistent Favorites panel.

Recently Opened

Automatically track assets opened during development and quickly return to them.

Most Used

See the assets you open most often and keep key project files within reach.

Quick Start

01
Open Project Hub

Open the Project Hub dockable editor window from the Unreal Engine Tools menu.

02
Search Assets

Type into the search bar to find Blueprints, Materials, Meshes, Maps, UI assets and more.

03
Open Assets

Double-click any search result to open it directly in the Unreal Editor.

04
Favorite Important Assets

Use the Favorite button to pin frequently used assets for faster access later.

Asset Filters

Project Hub includes a dropdown filter to narrow search results by common Unreal asset types.

  • All
  • Blueprints
  • Materials
  • Meshes
  • Textures
  • Animations
  • Audio
  • Niagara
  • UI
  • Maps
  • Data Assets
  • Input Assets
  • Sequences

Editor Integration

  • Dockable Unreal Engine editor window
  • Native Unreal asset icons
  • Persistent favorites and usage history
  • Double-click asset opening
  • Right-click removal from Favorites
  • Filters out engine content, redirectors and generated assets

Important Notes

  • Project Hub is an editor-only plugin.
  • It does not affect packaged builds.
  • Usage data is stored locally inside the project.
  • Currently tested and supported on Unreal Engine 5.7.
